# NexGen Counsel - JEE College Predictor

NexGen Counsel is a premium, AI-driven counseling tool designed to help JEE (Joint Entrance Examination) aspirants in India make informed decisions about their college choices.

## 🚀 Features

- **Massive Database**: Includes coverages for 30+ top institutions (IITs, NITs, IIITs, and GFTIs).
- **AI Recommendation Engine**: Recommends the best colleges based on your percentile, rank, and field of interest.
- **Probabilistic Analysis**: Calculates selection probability (Safe, Target, Reach, High Reach).
- **Side-by-Side Comparison**: Directly compare your "Dream College" with AI-suggested alternatives.
- **High-End UI**: Responsive glassmorphism design for a premium experience.

## 🛠️ Technology Stack

- **HTML5 / CSS3**: Vanilla implementation for maximum performance.
- **JavaScript**: Client-side logic for instant, privacy-focused calculations.
- **Google Fonts**: Utilizing "Outfit" for modern typography.
